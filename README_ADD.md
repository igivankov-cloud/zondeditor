# zondeditor

## Запуск (одной командой)
В корне проекта:

```bat
py run_zondeditor.py
```

Пока проект в переходном режиме: `run_zondeditor.py` запускает текущий рабочий монолит
`ZondEditor_SZ_v3_k2k4_dispatch_fix_params_step_depth_v5_nofreeze_k4fix_xlsxfix.py`.
Дальше будем выносить модули и постепенно переводить запуск на `src/zondeditor/ui/...`.
